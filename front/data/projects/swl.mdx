## Project Description

The main purpose of the Seaworld White Label project was to rebuild and modernize an old PHP website using Next.js 13. The new application offers improved performance, a more responsive user interface, and better maintainability compared to the original PHP site. My role as a developer in this project involved implementing various features, optimizing performance, and ensuring a seamless user experience.

<Box as="ul">
  <li>Next.js - Main framework for building the web application</li>
  <li>React - Main UI library</li>
  <li>TypeScript - Primary language for the project</li>
  <li>SASS - CSS preprocessor</li>
  <li>Firebase - Platform for user authentication, real-time updates, and other backend services</li>
  <li>Zustand - Lightweight state management library</li>
  <li>date-fns - Library for handling dates</li>
  <li>email-validator - Library for validating email addresses</li>
  <li>Frames React - Library for integrating with the Checkout.com payment system</li>
  <li>React Calendar - Library for rendering calendar components</li>
  <li>React Tabs - Library for creating tabbed interfaces</li>
  <li>React HTML Parser - Library for parsing and rendering HTML in React components</li>
  <li>DND Kit - Set of libraries for building drag-and-drop interfaces</li>
  <li>Jest - Main testing framework</li>
  <li>React Testing Library - Library for testing React components</li>
  <li>Testing Library User Event - Library for simulating user interactions</li>
  <li>MSW - Library for mocking API endpoints during development and testing</li>
  <li>Next Router Mock - Library for mocking Next.js router during testing</li>
</Box>
